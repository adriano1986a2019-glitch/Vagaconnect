# 🚀 Vagaconnect - Recrutamento por Vídeo

O **Vagaconnect** é uma solução moderna para o mercado de trabalho local. Focamos em eliminar a burocracia do primeiro contato através de tecnologia de **vídeo chamada integrada**.

## ✨ Diferenciais do Projeto
* 🎥 **Entrevistas Online:** Sistema de vídeo nativo para triagem rápida de candidatos.
* ⚡ **Eficiência Regional:** Focado em conectar o RH de empresas de Itapevi a talentos da região.
* 🎨 **Interface Moderna:** Design responsivo e intuitivo.

## 🛠️ Tecnologias
* HTML5 / Tailwind CSS
* JavaScript (ES6+)
* Jitsi Meet API (Vídeo chamadas)
